#### windows文件转换成linux文件
windows下的文件换行符是"\r\n"，而linux下文件换行符是"\n"
``` bash
# 切换方法
# windows文件转换成linux文件
sed "s/\r//g" filename
```

#### n和N的高级用法
sed处理文本是以行为单位处理的
n可以让sed暂存上一行的信息
N可以上sed暂存下一行的信息
