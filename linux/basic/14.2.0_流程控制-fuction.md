### 1. function语句
基本结构：
``` bash
function fun_name() {
执行语句1；
执行语句2；
...
}
## function里面调用的$1是fname()后面跟的第一个参数，需要和shell脚本的$1区分
```
用途：  
fuction主要是用来代码复用，减少重复代码量
