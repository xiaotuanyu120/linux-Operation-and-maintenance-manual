用if判断refer，并采取措施
问题描述：
查看网站访问日志，发现refer中有一个caww的网站，不断的导流量refer过来，量不大，但刚好对我们网站的访问造成了影响
解决办法：
在跟业务沟通过这个网站caww确实不是我们的合作业务后，采取以下措施
## 打开vhost配置文件，在server块中添加以下代码
**************************************
location / {
        if ($http_referer ~* (caww|another_one))
                {
                        return 403;
                }
}
**************************************
## 检查下配置语法错误 && 平滑重启服务
