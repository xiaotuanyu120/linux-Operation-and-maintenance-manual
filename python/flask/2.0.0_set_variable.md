### 2.0.0 set variables

---

### 1. 普通变量设定
``` python
# 设定变量
\{% set var = "var" %\}

# 使用变量
\{\{ var \}\}
```

---

### 2. 变量赋值给变量
``` python
# 传递变量给另外一个变量
\{% set var = request.path %\}

# 使用变量
\{\{ var \}\}
```
