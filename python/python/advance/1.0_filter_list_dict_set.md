### 1.0 筛选数据(列表、字典、集合)高级用法

---

### 1. 列表筛选数据
``` python
from random import randint

data = [randint(-10, 10) for _ in xrange(10)]

print data
[10, -6, 8, -10, -9, 7, -4, 6, 2, -2]

print filter(lambda x: x >= 0, data)
[10, 8, 7, 6, 2]

print [x for x in data if x >= 0]
[10, 8, 7, 6, 2]
```
